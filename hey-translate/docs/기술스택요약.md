# 사용된 기술 요약

## 설명

Hey! Translate 프로그램은 파이썬 기반으로 제작했습니다. 별도의 서버 없이 데스크탑 응용 프로그램으로만 구성되어 있습니다. STT(Speech To Text) 기능은 OpenAI사가 공개한 Whisper 오픈소스 기반으로 파생된 성능이 더 좋은 오픈소스 라이브러리인 Faster Whisper 기반으로 구성했습니다. 별도의 API 요청 없이 사용자 기기 장치에서 Whisper 모델을 작동시키는 것이기 때문에 STT 기능은 별도의 인터넷이 필요 없습니다. 마이크 입력은 SoundDevice 파이썬 모듈을 사용하여 STT 모델에 적합한 샘플 레이트로 적절히 변환하여 처리했습니다. 번역은 DeepL의 번역 API를 사용하여 인터넷 연결이 필요합니다. UI는 파이썬에서 유명한 TKinter UI 라이브러리 기반의 CustomTKinter 라이브러리를 사용했으며, 둘의 차이는 CustomTKinter가 디자인적으로 미려하다는 점이 있습니다.

## 정리

- 언어: 파이썬 (https://www.python.org)
- STT: Faster Whisper (https://github.com/SYSTRAN/faster-whisper)
- 마이크 입력: SoundDevice (https://github.com/spatialaudio/python-sounddevice)
- 번역: DeepL API (https://www.deepl.com/en/products/api)
- UI: CustomTkinter (https://customtkinter.tomschimansky.com)
